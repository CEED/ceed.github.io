gslib

This page is under construction

A library for Gather/Scatter-type nearest neighbor data exchanges for parallel spectral element computations.

Additional features include:

  • XXT solver (parallel direct solver)
  • AMG solver
  • Robust spectral element interpolation for a given set of points

gslib's applications include Nek5000 and Nektar++.

For more information, see the gslib GitHub repository: https://github.com/gslib/gslib.
