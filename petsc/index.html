PETSc

This page is under construction

PETSc is a scalable package for solving differential and algebraic equations. PETSc will coordinate with CEED to develop expressive interfaces for efficient and robust solution of the algebraic equations appearing in high-order/spectral element methods. This will include multilevel solvers that work with unassembled representations of linear operators.
